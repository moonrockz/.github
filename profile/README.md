# Moonrockz

Moonrockz is a GitHub organization dedicated to building **tooling and libraries for the [MoonBit](https://www.moonbitlang.com/) programming language**.

## What We Do

MoonBit is a modern, statically-typed programming language designed for WebAssembly. Moonrockz exists to grow and support the MoonBit ecosystem by providing:

- **Libraries** — reusable packages that extend MoonBit's standard capabilities, covering areas such as data structures, algorithms, parsing, and more.
- **Tooling** — developer tools that improve the MoonBit development experience, including build helpers, code generators, formatters, and testing utilities.
